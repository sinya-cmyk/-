# カレンダー型タスク管理アプリ

このプロジェクトは、カレンダー形式で日別タスクを管理できるシンプルなWebアプリです。

## 機能

- 月表示のカレンダー
- 日付クリックでタスクリストを表示
- タスクの追加、完了、削除
- ローカルストレージにタスクを保存
- ログイン機能とアカウント作成機能
- ユーザー別タスク管理

## 使い方

1. `index.html` をブラウザで開きます。
2. カレンダーの左右ボタンで月を切り替えます。
3. 日付をクリックしてタスクを追加します。
4. タスクの完了 / 削除ができます。

## ファイル構成

- `index.html` - アプリの構造
- `styles.css` - 見た目のデザイン
- `app.js` - カレンダーとタスク管理のロジック
